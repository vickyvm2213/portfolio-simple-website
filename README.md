# Personal Portfolio Website

This is a simple personal portfolio website built using HTML and CSS. The website consists of two pages: a Home page and a Contact page.

The Home page introduces me and highlights my skills and basic information, while the Contact page allows visitors to get in touch with me.

## Features
- Two-page website (Home & Contact)
- Clean and simple layout
- Easy navigation between pages
- Structured using semantic HTML
- Styled with basic CSS

## Technologies Used
- HTML5
- CSS3

## Purpose
This project was created to practice and improve my front-end development skills, focusing on building structured web pages and styling them using CSS.
